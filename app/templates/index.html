World Happiness Project

Maps

    All Years 2015 2016 2017 2018 2019
Each map displays up to 157 countries. A circle represents each country by Happiness Rank according to color.

Rank 1-10: Green
Rank 11-25: Yellow
Rank 26-50: Orange
Rank 51-100: Red
Rank 101-157: Brown

The 'All Years' map displays circle size according to Happiness Rank over each year.

Each subsequent map represents each country differently by size according to a selectable factor.
I.e. GDP (Per Capita), Life Expectancy, and Government Trust.

This project was created by Jonathan Arp, Gabe Cweigenberg, Jesse DeLaCruz, Nicolette Keplinger, and Dylan Kurbel.
